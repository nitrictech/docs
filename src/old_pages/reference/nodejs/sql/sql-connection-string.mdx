# Node.js - sql.connectionString()

Returns the connection string for a SQL database at runtime.

```javascript
import { sql } from '@nitric/sdk'

const db = sql('my-data')

// Should be called at runtime, such as in a service handler
const connStr = await db.connectionString()
```
