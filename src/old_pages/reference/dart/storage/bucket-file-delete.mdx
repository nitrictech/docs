# Dart - bucket.file.delete()

Delete a file from a bucket.

```dart
import 'package:nitric_sdk/nitric.dart';

final assets = Nitric.bucket("assets").allow([
  BucketPermission.delete,
]);

final logo = assets.file("images/logo.png");

await logo.delete();
```
