# Node.js - sql()

Creates a reference to a SQL database.

```javascript
import { sql } from '@nitric/sdk'

const db = sql('my-data')
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The unique name of this database within the project.
  </Property>
</Properties>
