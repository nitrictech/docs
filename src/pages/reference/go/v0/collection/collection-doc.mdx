# Go - collection.doc()

Get a reference to a document in the collection.

```go
import (
	"fmt"

	"github.com/nitrictech/go-sdk/resources"
)

func main() {
	profiles, err := resources.NewCollection("profiles").With(resources.CollectionReading, resources.CollectionWriting)
	if err != nil {
		return
	}

	drakeProfileRef := profiles.Doc("Drake Mallard")

	if err := resources.Run(); err != nil {
		fmt.Println(err)
	}
}
```

## Parameters

<Properties>
  <Property name="key" required type="string">
