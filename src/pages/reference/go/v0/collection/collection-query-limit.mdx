# Go - collection.query.limit()

Limit the number of results returned by a query.

```go
import (
	"fmt"

	"github.com/nitrictech/go-sdk/resources"
)

func main() {
	profiles, err := resources.NewCollection("profiles").With(resources.CollectionReading, resources.CollectionWriting, resources.CollectionDeleting)
	if err != nil {
		fmt.Println(err)
		return
	}

	profilesQuery := profiles.Query().Limit(10)

	if err := resources.Run(); err != nil {
		fmt.Println(err)
	}
}
```

## Parameters

<Properties>
  <Property name="limit" required type="int">
    The limit to apply to the query.
