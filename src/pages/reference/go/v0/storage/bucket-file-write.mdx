# Go - bucket.file.write()

Write a file to a bucket.

```go
import (
	"context"
	"fmt"

	"github.com/nitrictech/go-sdk/resources"
)

func main() {
	bucket, err := resources.NewBucket("bucket-name").With(resources.BucketWriting)
	if err != nil {
		return
	}

	err := bucket.File("cat.png").Write(context.TODO(), []byte("contents"))
	if err != nil {
		return
	}

	if err := resources.Run(); err != nil {
		fmt.Println(err)
	}
}
```

## Parameters

<Properties>
  <Property name="ctx" required type="context">
    The context of the call, used for tracing.
  </Property>
  <Property name="data" required type="[]byte">
    The data to write to the file.
  </Property>
</Properties>
