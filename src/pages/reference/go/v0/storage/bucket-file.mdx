# Go - bucket.file()

Create a reference to a file within a bucket.

```go
import (
	"fmt"

	"github.com/nitrictech/go-sdk/resources"
)

func main() {
	bucket, err := resources.NewBucket("bucket-name").With(resources.BucketReading)
	if err != nil {
		fmt.Println(err)
		return
	}

	file := bucket.File("cat.png")

	if err := resources.Run(); err != nil {
		fmt.Println(err)
	}
}
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The unique name/reference to the file.
  </Property>
</Properties>

### Notes

The file does not need to exist, only a reference to that file is being created.
