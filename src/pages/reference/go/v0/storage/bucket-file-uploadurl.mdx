# Go - bucket.file.getUploadUrl()

Create an upload URL for a file within a bucket.

```go
import (
	"context"
	"fmt"

	"github.com/nitrictech/go-sdk/resources"
)

func main() {
	bucket, err := resources.NewBucket("bucket-name").With(resources.BucketWriting)
	if err != nil {
		fmt.Println(err)
		return
	}

	uploadUrl, err := bucket.File("cat.png").UploadUrl(context.TODO())
	if err != nil {
		fmt.Println(err)
		return
	}

	if err := resources.Run(); err != nil {
		fmt.Println(err)
	}
}
```

## Parameters

<Properties>
  <Property name="ctx" required type="context">
    The context of the call, used for tracing.
  </Property>
  <Property name="expiry" optional type="int">
    Seconds until link expiry. Maximum of `604800` (7 days).
  </Property>
</Properties>
