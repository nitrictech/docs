# Go - topic()

Creates a new Topic.

```go
import (
	"fmt"

	"github.com/nitrictech/go-sdk/resources"
)

func main() {
	updates := resources.NewTopic("updates")

	if err := resources.Run(); err != nil {
		fmt.Println(err)
	}
}
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The name of the topic to create.
  </Property>
</Properties>
