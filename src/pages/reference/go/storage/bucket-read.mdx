# Go - Bucket.Read()

Read the contents of a file from a bucket.

```go
import (
  "context"
  "fmt"

  "github.com/nitrictech/go-sdk/nitric"
)

func main() {
  bucket, err := nitric.NewBucket("bucket-name").Allow(nitric.BucketRead)
  if err != nil {
    return
  }

  contents, err := bucket.Read(context.TODO(), "cat.png")
  if err != nil {
    return
  }

  fmt.Println(contents)

  if err := nitric.Run(); err != nil {
    fmt.Println(err)
  }
}
```

## Parameters

<Properties>
  <Property name="ctx" required type="context">
    The context of the call, used for tracing.
  </Property>
  <Property name="key" required type="string">
    The key of the file to read
  </Property>
</Properties>
