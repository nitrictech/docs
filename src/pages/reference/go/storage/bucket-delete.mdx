# Go - Bucket.File.Delete()

Delete a file from a bucket.

```go
import (
  "context"
  "fmt"

  "github.com/nitrictech/go-sdk/nitric"
)

func main() {
  bucket, err := nitric.NewBucket("bucket-name").Allow(nitric.BucketDelete)
  if err != nil {
    return
  }

  err = bucket.Delete(context.TODO(), "cat.png")
  if err != nil {
    return
  }

  if err := nitric.Run(); err != nil {
    fmt.Println(err)
  }
}
```

## Parameters

<Properties>
  <Property name="ctx" required type="context">
    The context of the call, used for tracing.
  </Property>
  <Property name="key" required type="string">
    The key of the file to delete.
  </Property>
</Properties>
