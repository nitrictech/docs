# CLI Installation

## Prerequisites

Nitric relies on functionality from the following projects to help retrieve plugins, containerize and deploy your application.

Please follow these links to the official installation steps for each.

- [Git](https://git-scm.com/)
- [Docker](https://docs.docker.com/get-docker/)
- [Pulumi](https://www.pulumi.com/reference/cli/)

## Installing the Nitric CLI

<CodeGroup>

```bash {{ title: 'macOS', label: 'nitric is available via <a href="https://brew.sh/">homebrew</a>' }}
brew install nitrictech/tap/nitric
```

```bash {{ title: 'Windows', label: 'nitric is available via <a href="https://scoop.sh/">scoop</a>' }}
scoop bucket add nitric https://github.com/nitrictech/scoop-bucket.git
scoop install nitric
```

```bash {{ title: 'Linux', label: 'nitric can be downloaded as a scripted install via curl.' }}
curl https://nitric.io/install | bash
```

```bash {{ title: 'Brew', label: 'nitric is available via <a href="https://brew.sh/">homebrew</a>' }}
brew install nitrictech/tap/nitric
```

</CodeGroup>

### Manual Install

You can download pre-compiled binaries from the [releases](https://github.com/nitrictech/cli/releases) page and copy them into your desired location

## Update to the latest version

<CodeGroup>

```bash {{ title: 'macOS' }}
brew upgrade nitric
```

```bash {{ title: 'Windows' }}
scoop update nitric
```

```bash {{ title: 'Linux' }}
curl https://nitric.io/install | bash
```

```bash {{ title: 'Brew' }}
brew upgrade nitric
```

</CodeGroup>

You can easily check which version you're on and if you're up to date by running `nitric version`.
