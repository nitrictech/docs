# .NET - Bucket.File.Delete()

Delete a file from a bucket.

```csharp
using Nitric.Sdk.Resource;

var assets = Nitric.Sdk.Nitric.Bucket("assets").With(BucketPermission.Deleting);

var logo = assets.File("images/logo.png");

logo.Delete();

Nitric.Sdk.Nitric.Run();
```
