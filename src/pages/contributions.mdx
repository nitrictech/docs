# Contributions

Thanks for your interest in contributing to the Nitric community. There are plenty of ways to get involved, we've summarized a few here to get you started.

Nitric is Open-Source, our team encourages you to ask questions, raise requests for new features and let us know about bugs! If you would like to know more about our future plans or what we are currently working on, you can look at the [Nitric roadmap](https://github.com/orgs/nitrictech/projects/4).

## Questions and discussions

You can ask questions and launch discussions about Nitric-related topics in the nitric repository on [GitHub](https://github.com/nitrictech/nitric/discussions). A [Discord](https://discord.gg/Webemece5C) server is also available if you'd like to chat about features, issues or connect with others using Nitric.

## Create a bug report or feature request

If you encounter an error or run into issues while using Nitric, please make sure to create a [bug report](https://github.com/nitrictech/nitric/issues/new?assignees=&labels=&template=bug_report.md&title=%27Create%20bug%20report%27). If you'd like to suggest a change or request new functionality you can submit a [feature request](https://github.com/nitrictech/nitric/issues/new?assignees=&labels=&template=feature_request.md&title=%27Submit%20feature%20request%27).

You can also check out our [Creating bug reports](./support) guide to step you through the process.
