# Python - websocket()

Creates a new Websocket API.

```python
from nitric.resources import websocket

public_websocket = websocket("public")
```

## Parameters

<Properties>
  <Property name="name" required type="string">
    The unique name of this Websocket within the app. Subsequent calls to
    `websocket` with the same name will return the same object.
  </Property>
</Properties>

---

## Examples

### Create a Websocket

```python
from nitric.resources import websocket

chat_websocket = websocket("chat")
```
