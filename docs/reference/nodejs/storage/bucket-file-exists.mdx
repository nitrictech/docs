# Node.js - bucket.file.exists()

Determine if a file exists in a bucket.

```javascript
import { api, bucket } from '@nitric/sdk'

const assets = bucket('assets').allow('read')

api('example').get('/', (ctx) => {
  const exists = await assets.file('images/logo.png').exists()
});
```
