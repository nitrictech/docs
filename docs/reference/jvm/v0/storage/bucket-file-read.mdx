# JVM - bucket.file.read()

Read the contents of a file from a bucket.

<CodeWithTabs>

```java !!tabs Java
import io.nitric.Nitric;
import io.nitric.resources.BucketPermission;

public class Application {
  public static void main(String[] args) {
    var bucket = Nitric.IN`STANCE.bucket("images").with(BucketPermission.Read);

    var contents = bucket.file("cat.png").read();

    Nitric.INSTANCE.run();
  }
}
```

```kotlin !!tabs Kotlin
import io.nitric.Nitric
import io.nitric.resources.BucketPermission

fun main() {
  val bucket = Nitric.bucket("images").with(BucketPermission.Read)

  val contents = bucket.file("cat.png").read()

  Nitric.run()
}
```
