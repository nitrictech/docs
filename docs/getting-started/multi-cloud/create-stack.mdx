The Nitric CLI provides helper commands to create **stacks** and the resources that define them. Using the `make:stack` command, will guide you through the creation of an initial Nitric stack.

## Create a new Stack

The `nitric make:stack` command will allow you to name your stack and choose a `Service template` in your preferred language:

```bash
nitric make:stack nitric-tutorial
```

### Choose a service template
